# VanillaBot

A simple Whatsapp bot that uses the Baileys library. It's designed to be light-weight, fast and modular, allowing for easy command installation and creation.

## Developing

For developers, check out the API Documentation [here](api-docs/main.md).

## Setup

Clone the repository.
```bash
git clone https://github.com/nataniss/vanilla/
cd vanilla
```

Install dependencies, then start the bot.
```bash
npm i
npm run vanilla
```

A QR Code will appear. Scan it from Whatsapp's client on your phone.
